当下:智能体走向垂直,并开始「做决策」;本地优先成反向副线Present: agents go vertical and start “making decisions”; local-first as a counter-current

当下最强的信号不是「又一个 harness / 又一份技能」,而是 智能体开始在具体领域里产出成果、甚至做出有后果的决策,且证据横跨多窗口:(1) 金融决策——当日 ai-berkshire(+1,386,价值投资研究框架)与 Vibe-Trading(+839,个人交易 agent),把巴菲特/芒格式方法论交给 agent 执行;(2) 多模型议会——council-of-high-intelligence(+331)用 18 个人设跨多家 LLM 互相质询,专治高风险决策里的幻觉;(3) 创意生产——video-use(+967)、本月 MoneyPrinterTurbo(+26,117)与周/月双榜的 OpenMontage 把视频生产 agent 化;(4) 支撑这些垂直应用的上下文/记忆层仍在高位:本周 cognee(AI 记忆平台,+6,335)、codebase-memory-mcp(+9,899)、Agent-Reach(+7,928)、design.md(+7,104)。与此同时出现一条明显的反向副线——本地优先:simplex-chat(#2,无任何用户标识符)、FluidVoice(#7,完全离线听写)与 openpilot(#8,车端本地计算)同日在榜。The strongest present-day signal isn't “yet another harness / yet another skill pack” — it's that agents are starting to produce outputs and even make consequential decisions in specific domains, with evidence spanning windows: (1) financial decisions — today's ai-berkshire (+1,386, value-investing research) and Vibe-Trading (+839, personal trading agent) hand Buffett/Munger-style methodology to an agent; (2) multi-model deliberation — council-of-high-intelligence (+331) has 18 personas cross-examine each other across multiple LLMs to fight hallucination in high-stakes calls; (3) creative production — video-use (+967), this month's MoneyPrinterTurbo (+26,117) and the week/month dual-charting OpenMontage turn video production agentic; (4) the context/memory layer feeding these verticals stays high: this week's cognee (AI memory platform, +6,335), codebase-memory-mcp (+9,899), Agent-Reach (+7,928), design.md (+7,104). At the same time, a clear counter-current — local-first — shows up on the same day: simplex-chat (#2, no user identifiers), FluidVoice (#7, fully-offline dictation), and openpilot (#8, on-device compute).

未来:垂直智能体需要「可验证战绩」;本地优先继续抬升Future: vertical agents need verifiable track records; local-first keeps rising

从今天的信号外推,给出可证伪的预测:(1) 垂直决策 agent 需要「可验证战绩」——投资/交易类 agent(ai-berkshire、Vibe-Trading)只有公开回测、实盘曲线与风控才能站稳;预测未来一个季度,trending 上会出现更多「agent + 回测 / 评测 / 审计」类仓库,光有方法论的会快速退潮。(2) 多智能体「议会 / 辩论」成为高风险决策标配——council 这类「多模型互相质询」的模式会被嵌入投资、医疗、法务等场景以压制幻觉。(3) 本地优先 / 端侧 AI 持续抬升——隐私与成本双重压力下,simplex-chat、FluidVoice、openpilot 这类「数据不出本地」的项目会更频繁登榜;预测两周内日榜至少再出现 1 个端侧/离线 AI 应用。(4) 基础设施层(技能/harness/记忆)走向「水电煤」——superpowers、ECC、cognee、codebase-memory-mcp 这些会沉淀为被默认依赖的底座,价值与注意力上移到垂直应用与专有领域数据。可证伪:未来两周,「垂直 agent 应用(尤其金融/创意)+ 本地优先」在日榜的占比继续上升,而纯技能/prompt 合集继续从 #1 退场。Extrapolating from today's signals, here are falsifiable predictions: (1) vertical decision agents will need “verifiable track records” — investing/trading agents (ai-berkshire, Vibe-Trading) only hold up with public backtests, live-equity curves and risk controls; expect more “agent + backtest / eval / audit” repos next quarter, while methodology-only ones fade fast. (2) multi-agent “council / debate” becomes standard for high-stakes decisions — patterns like council (models cross-examining each other) get embedded into investing, healthcare and legal to suppress hallucination. (3) local-first / on-device AI keeps rising — under privacy and cost pressure, “data never leaves the device” projects like simplex-chat, FluidVoice and openpilot chart more often; predict at least one more on-device/offline AI app on the daily board within two weeks. (4) the infrastructure layer (skills/harness/memory) becomes “utilities” — superpowers, ECC, cognee, codebase-memory-mcp settle into a default-depended substrate, while value and attention move up to vertical apps and proprietary domain data. Falsifiable: over the next two weeks, “vertical agent apps (esp. finance/creative) + local-first” rise as a share of the daily board, while pure skill/prompt collections keep leaving the #1 spot.

The evidence is that these repos dominated the charts for months — the protagonists were "model weights + orchestration frameworks."
