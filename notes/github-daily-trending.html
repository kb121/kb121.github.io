当下:两层并存 —— 头部是「Agent 淘金」,日常 trending 是「常青基建」Present: two coexisting layers — the head is an "agent gold-rush," daily trending is "evergreen infra"

把五个窗口叠在一起,会看到两层并存的结构。头部(半年/一年)是 Agent 淘金:10/10 都在 agent 生态,且已长出「供应链」分工 —— 技能(anthropics/skills)→ harness(ECC、spec-kit)→ 元 harness(本周 omnigent:在 Claude Code/Codex/Pi 之上抽公共层、可换可组合)→ 审查/安全(本月 alibaba/open-code-review、安全技能 defending-code-reference-harness,以及一周榜上的 AUR 恶意包扫描)。降本也成显学:本周 shadcn/improve「贵模型规划、便宜模型执行」、MiMo-Code 自带免费通道。而当天 trending 却是另一层:常青基建 —— iptv、freeCodeCamp、VoxCPM(TTS)、iroh(P2P 网络)、meshery(云原生)无一是 agent。证据很清楚:每天给 star 的大众 仍在追教育/媒体/网络/多模态等常青项目,而最大的新仓库 几乎全被 agent 淘金占据 —— 两层真实并存,互不取代。 Stacking the five windows reveals two coexisting layers. The head (6mo/year) is the agent gold-rush: 10/10 in the agent ecosystem, already with a "supply-chain" division of labor — skills (anthropics/skills) → harness (ECC, spec-kit) → meta-harness (this week's omnigent: a common layer over Claude Code/Codex/Pi, swap-or-combine) → review/security (this month's alibaba/open-code-review, the security skill defending-code-reference-harness, plus AUR-malware scanning on the weekly board). Yet today's trending is the other layer: evergreen infra — iptv, freeCodeCamp, VoxCPM (TTS), iroh (P2P networking), meshery (cloud-native), none of them agents. The evidence is clear: the broad crowd handing out daily stars still follows evergreen education/media/networking/multimodal projects, while the biggest new repos are almost entirely agent gold-rush — two real layers coexisting, neither replacing the other.
